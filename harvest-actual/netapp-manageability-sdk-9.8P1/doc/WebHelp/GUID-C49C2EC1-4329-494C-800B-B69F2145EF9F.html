OpenSSL Library Information
NetApp Manageability SDK provides core libraries and OpenSSL libraries. NMSDK 9.8P1 is using OpenSSL 1.0.2w, which is FIPS compliant and it is using openssl-fips-2.0.16 module.
You can find the OpenSSL static libraries for various platforms at the below paths:
Platform
OpenSSL Version
Library Details
Windows
1.0.2w
libeay32.lib
ssleay32.lib
The libraries for the 32-bit platform are at lib\nt\.
The libraries for the 64-bit platform are at lib\nt\x64\.
Linux, FreeBSD, or PowerPC
1.0.2w
libcrypto.a
libssl.a
The library details for different operating systems are as follows:
•
Linux 32-bit libraries are at lib\linux.
•
Linux 64-bit libraries are at lib\linux-64.
•
FreeBSD libraries are at lib\freebsd-amd64.
•
PowerPC libraries are at lib\ppc64-redhat-linux.
