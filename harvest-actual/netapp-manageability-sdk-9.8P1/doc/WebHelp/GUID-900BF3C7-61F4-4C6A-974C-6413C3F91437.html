Building SDK for Windows platform
Before you begin 
•
You must ensure that the following binaries are present in NetApp Manageability SDK zip at \lib\nt\ and \lib\nt\x64 locations for Windows 32-bit and Windows 64-bit respectively:
◦
libeay32.dll, ssleay32.dll (dynamic-link library)
◦
libeay32.lib, ssleay32.lib (import library)
•
You must ensure that the following libraries for the Windows 32-bit and Windows 64-bit platforms are located at \lib\nt\ and \lib\nt\x64, respectively:
◦
libnetapp.lib, libadt.lib (static library)
◦
libxml.lib (import library)
◦
libxml.dll (dynamic linking library)
Note: zephyr.dll is not supported in SDK 5.5. During the runtime, SDK 5.5 provides an additional library (libxml.dll) along with the libeay32.dll and ssleay32.dll libraries.
Steps 
1.
Replace the zephyr.lib library file with the libnetapp.lib, libadt.lib, libxml.lib, libeay32.lib, ssleay32.lib library files, and then compile the code.
